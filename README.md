# Campus Life Super App

## Description
This project is a multi-page web application designed to help students navigate campus life. It includes pages for events, dining, and a campus map.

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

## Features
- Multi-page navigation (Home, Events, Dining, Map)
- Responsive layout using Bootstrap
- Interactive buttons using JavaScript
- API placeholder for future data integration

## How to Use
Open the deployed link and use the navigation bar to explore different sections of the app.

## Future Improvements
- Connect a real API for live event or dining data
- Improve styling and layout
- Add interactive map functionality
